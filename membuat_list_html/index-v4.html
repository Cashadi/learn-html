Di Web Ez Codefirst kita akan mempelajari 2 bahasa pemograman diatas, yaitu PHP dan Javascript. PHP (Hypertext Pre-processor) adalah bahsa pemograman yang umum digunakan untuk membuat sebuah website, sedangkan Javascript saat ini menjadi salah satu bahasa yang paling populer di dunia. Peranan PHP sendiri akan kita gunakan untuk mengatur/memproses data melalui server, dan Javascript akan kita gunakan mulai dari sisi fungsional pada website dan mengatur interaksi website yang akan kita buat.
