Cara Menggunakan WhatsApp Web

Menggunakan WhatsApp Web bukan berarti kamu akan membuat akun WhatsApp yang baru. Untuk menggunakan fitur berbasis komputer ini, terlebih dahulu kamu harus memiliki aku WhatsApp yang tersedia di smartphone. Nantinya WhatsApp web akan melakukan sinkronasi dari akun WhatsApp di smartphone ke WhatsApp Web.

Berikut akan diulas terlebih dahulu cara login WhatsApp Web :

  1. Buka Browser, lalu kunjungi situs WhatsApp di web.whatsapp.com.
  2. Buka aplikasi WhatsApp di smartphone, lalu klik simbol titik tiga yang berada di pojok kanan atas. Kemudian klik WhatsApp Web.
  3. Selanjutnya akan ada semacam tutorial untuk scan QR Code di WhatsApp Web. Lalu klik OK. Got It.
  4. Pindai QR Code yand ada di WhatsApp Web dengan menggunakan fitur scan yang ada di aplikasi WhatsApp. Jika pindai QR Code berhasil, akun WhatsApp di smartphone otomatis akan tersinkronasi dengan WhatsApp Web.

Sudah tahu cara login, sekarang saatnya mengetahui cara log out dari WhatsApp Web berikut tutorialnya :

  1. Untuk WhatsApp Web, klik tombol titik tiga yang berada di pojok kanan atas kolom chats. Kemudian pilih menu Log Out
  2. Untuk WhatsApp di smartphone, pertama klik tombol titik tiga yang berada di pojok kanan atas, Kemudian pilih WhatsApp Web.
  3. Selanjutnya akan terpampang perangkat apa saja yang sedang tersinkronasi dengan akun WhatsApp kamu. Kemudian pilih perangkat mana yang ingin WhatsApp Web di log out.
  4. setelah itu akan ada pertanyaan apakah ingin keluar dari perangkat ini, lalu pilih Log Out

Itu dia cara menggunakan WhatsApp Web untuk log in dan log out akun. Semoga artikel ini membantu.
